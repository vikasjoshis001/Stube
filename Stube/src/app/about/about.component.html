About


  • This is the Web Application project which provides many free Technical courses from YouTube at single place.It is developed in Angular 9.It is currently live on google firebase cloud.

  • This Web Application is useful because most of the times students search courses on Youtube and they do not found a suitable course which they should learn.

  • This Application provides every single information of every course like Lecturer of course,Language of course,Rating of course,Total Lectures and many more

  • In this Web Application there is no issue of Copyright.As when user selects specific course.It will redirect to YouTube.
